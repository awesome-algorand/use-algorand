> API endpoints for algod operations.

### Structure

`Algodv2` is provided by the `use-wallet` package, which is used to interact with the Algorand blockchain.
All `query` packages are built on top of the `core` package, which provides the `QueryOptions`.
This ensures that the query options are consistent across all packages.

| Package                                                                                                            | Description    |
|--------------------------------------------------------------------------------------------------------------------|----------------|
| [@awesome-algorand/query-core](https://github.com/awesome-algorand/use-algorand/tree/main/packages/query-core)     | QueryOptions   |
| [@awesome-algorand/react-query](https://github.com/awesome-algorand/use-algorand/tree/main/packages/react-query)   | Used in React  |
| [@awesome-algorand/svelte-query](https://github.com/awesome-algorand/use-algorand/tree/main/packages/svelte-query) | Used in Svelte |
