> **GET** /v2/ledger/supply

#### Get the current supply reported by the ledger.

            Using the query-core library directly is not recommended!
        </Aside>
        #### Install
        ```bash
        npm install @awesome-algorand/query-core @txnlab/use-wallet-react @tanstack/react-query algosdk --save
        ```
        #### Usage
        ```jsx
        import { useQuery } from 'react-query';
        import { getSupply } from '@awesome-algorand/query-core';
        import { useWallet } from "@txnlab/use-wallet-react";

        export function SupplyViewer() {
            const walletManager = useWallet();
            const query = useQuery(getSupply());
        }
        ```
