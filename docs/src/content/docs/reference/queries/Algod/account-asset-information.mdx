> **GET** /v2/accounts/**\{address\}**/assets/**\{assetId\}**?format=**\{format\}**

#### Get account information about a given asset.

Given a specific account public key and asset ID, this call returns the account’s asset holding and asset parameters (if either exist).
Asset parameters will only be returned if the provided address is the asset’s creator.

### Properties

| Name         | Type                  | Description                                                                                               |
|--------------|-----------------------|-----------------------------------------------------------------------------------------------------------|
| **address*** | `string`              | An account public key                                                                                     |
| **assetId*** | `number`              | An asset identifier                                                                                       |
| **format**   | `"json" \| "msgpack"` | Configures whether the response object is JSON or MessagePack encoded. If not provided, defaults to JSON. |

\* indicates required property
