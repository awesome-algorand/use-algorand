> **GET** /v2/status/wait-for-block-after/**\{round\}**

#### Gets the node status after waiting for a round after the given round.

Waits for a block to appear after round \{round\} and returns the node's status at the time.
There is a 1 minute timeout, when reached the current status is returned regardless of whether or not it is the round after the given round.

### Properties

| Name       | Type     | Description                              |
|------------|----------|------------------------------------------|
| **round*** | `string` | The round to wait until returning status |


\* indicates required property
