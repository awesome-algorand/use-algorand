## Application Boxes

Let's look at the preview on the bottom right, we'll notice that it is not displaying anything.
This is because we haven't yet implemented the `useApplicationBoxes` hook in our React application.

To address this, we'll call the `useApplicationBoxes` hook to handle the fetch call in the editor on the top right.


```tsx add={5}
import { useAccountInformation } from '@awesome-algorand/react-query';

export function ApplicationBoxesViewer() {
    const data = {applicationId: 62368684}
    const query = {isLoading: true, isError: false, error: undefined, data: undefined};

    if(query.isLoading) return <div>Loading...</div>
    if(query.isError) return <div>Error: {query.error.message}</div>

    return (
        <div>
            <h2>Application: {query.data.id || "Unknown"}</h2>
            <h2>Creator: {query.data.params.creator || "Unknown"}</h2>
        </div>
    )
}
```
